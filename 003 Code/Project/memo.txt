작업 목록
타이머: 
	O 15분 (11 : 11 형식, 한자리 수 초일때는 09, 08, 07 형식으로 표시되게)
	O 전파탑 하나 지을때마다 30초 줄어들게
	O 철거 시 20초 복구

전파탑:
	O 스파이는 전파탑에서 통신 가능, 
	O 그 행동을 하는 동안 시간 감소 속도 2배
	O 최대 한 전파탑 당 20초씩만(40초 감소)
	O 시간 흐르는 속도 가속하는 동안 게이지 보이게
	O 일정 개수만큼 만들면(파괴 안 되고 맵에 존재) 즉시 승리
		철로 제작 가능

댐: 
	O 구역 5개, 모두 만들면 일반 비버 승리
	O 창고와 자기 자신의 자원이 충분하면 만들기 시작
	O 댐마다 필요 자원 다름, 총합 같고 그 안에서 각각 랜덤
	O 게이지 생기고 천천히 올라감
	O 그동안 작업(기본속도(1) +1 = 2)가능
	O 반대로 방해도 가능(기본속도(1) -1.5 = -0.5)
	즉 기본 속도가 1일때 작업과 방해가 동시에 이뤄지면 0.5
	방해만 한다면 -0.5, 작업만 한다면 2
	O 방해로 인해 게이지가 0이 된다면 댐 파괴

채집: 
	O 따로 화면에 띄움
	X 키보드 조작(타이밍 게임)
	X 채집 속도 올려주는 템 존재(도끼, 곡괭이, 삽 등)
	X 돌 -> 나무 곡괭이 필요, 철 -> 돌 곡괭이 필요
	X 도끼 -> 나무 채집 속도 증가
	X 삽 -> 진흙 채집 속도 증가

창고: 
	O 창고에 자원 넣기, 넣을때 수량 조절 가능
	O 넣은건 못 빼고 댐 만들 때만 사용 가능

인벤토리: 
	O 아이템 옮기기
	O 아이템 땅에 떨어뜨리기
	O 아이템 버리기 
	O 아이템 줍기
	O 아이템 여러개일때 나누기, 합치기
	O 아이템 장착, 해제 가능(모자, 도구 +옷(우선순위 낮음))
	팰월드처럼
	O 장착한 아이템 화면에 나오도록 하기

감옥: 
	O 가두기 도구 존재(제작 난이도 높음)
	O 30 -> 40 -> 50초 갖힐 때마다 갖혀있는 시간 증가
	O 시간이 얼마 안 남았을 때(2분) 스파이는 본인 정체를 밝히고
	긴급 탈출 1회 가능(빨간 표시 같은거 생김)
	O 시간 지난 후 or 긴급 탈출 후 맵을 구획별로 나누고 각각의
	X 지역의 사람 수를 표시해준
        O 지도를 보고 탈출 지역 선택
	O 매우 비싼 도구, 탈옥 열쇠 미리 만들어 두면 탈출 가능

도구 제작: 
	O 가공소에서 제작 가능, 팰월드처럼
	
	곡괭이(나무, 돌, 철), 도끼(나무, 돌, 철), 삽(나무, 돌, 철), 
	모자(작업속도 증가)
	신발?(이속 증가)
	감옥 가두는 밧줄, 감옥 탈출 열쇠
